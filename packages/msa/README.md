# bio-analyze-msa

Multiple Sequence Alignment and Phylogenetic Tree module for the bio-analyze toolbox.

## Features

- Multiple Sequence Alignment using standard tools (MAFFT, MUSCLE) or a pure Python fallback.
- Phylogenetic tree construction (Distance-based via Biopython, Maximum Likelihood via FastTree).
- Integration with `bio-analyze-core` and Typer CLI.
